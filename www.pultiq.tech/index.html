Funkcie

Všetko, čo moderný support potrebuje

Jeden systém pre tickety, chat, knowledge base, SLA a automatizácie — bez integračného pekla.

🤖

AI asistent (Claude)

Automatická kategorizácia, návrhy odpovedí a sumarizácia vlákien. Haiku pre rýchle operácie, Sonnet pre komplexné.

🏢

Multi-tenancy

Shared databáza s tenant_id a Row-Level Security politikami. Žiadny únik dát medzi klientmi.

💬

Live chat widget

Embeddovateľný widget so Shadow DOM — neovplyvní štýly vášho webu. Agent dashboard v reálnom čase.

⏱️

SLA engine

Politiky s breach countdownom, eskalácie a notifikácie. Reportovanie compliance v reálnom čase.

⚡

Automatizácie

Trigger / condition / action pravidlá. Makrá pre opakované akcie. Workflow bez kódu.

📚

Knowledge base

Full-text vyhľadávanie, kategórie, verejné aj interné články. AI navrhuje relevantné pri tikete.

📊

Analytika

Resolution time, response time, SLA compliance, trendy podľa agenta, tímu a kategórie.

📧

Email integrácia

Postmark inbound webhooks, outbound notifikácie, threading. Email-to-ticket out of the box.

🔒

Security first

Rate limiting, XSS sanitizácia, auth middleware, security headers, encrypted attachments cez R2.

Workflow

Od ticketu k vyriešeniu za minúty

01

Príjem

Email, widget, portál alebo API. Všetky kanály v jednom inboxe.

02

AI klasifikácia

Claude priradí kategóriu, prioritu a navrhne riešenie z KB.

03

Automatizácia

Pravidlá smerujú ticket správnemu agentovi, spúšťajú SLA.

04

Vyriešenie

Agent pošle odpoveď, zákazník dostane email, analytika zachytí.
